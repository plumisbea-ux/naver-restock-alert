# 재입고 알림 데모 페이지

GitHub Pages용 데모 페이지입니다.

## 반영 내용

- 스토어명: `AnDre`
- 스토어명은 텍스트가 아니라 `assets/andre-logo.svg` 이미지로 표시
- 상단 스토어명 이미지와 카테고리 버튼은 모두 메인 상품 목록으로 이동
- 페이지 타이틀: `재입고 알림 데모 페이지`
- 상품 2번/9번 포함 모든 상품 이미지를 로컬 SVG로 제공해 빈 이미지 방지
- 모든 상품에 공통으로 품절 옵션 포함
- 품절 옵션 표시: `차콜헤더 : 재입고 알림은 톡톡에서 (품절)`
- 품절 옵션은 선택 박스에 담기지 않음
- 품절 옵션 클릭 시 기존 스마트스토어처럼 구매 불가 alert 표시
- alert 문구: `품절인 옵션은 구매하실 수 없습니다. 재입고 문의는 톡톡에서 가능합니다.`
- 재입고 알림 전용 버튼은 제거
- `톡톡문의` 버튼을 누르면 실제 톡톡 페이지 탭 오픈
- 기존 Vercel Webhook이 있으면 open 이벤트 mock 호출
