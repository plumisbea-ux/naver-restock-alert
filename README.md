# 재입고 알림 데모 페이지

GitHub Pages에 바로 올릴 수 있는 정적 데모입니다.

## 반영 사항

- 상품 이미지를 로컬 SVG가 아니라 인터넷 이미지 URL로 사용했습니다.
- 메인 페이지의 `4점 이상 리뷰가 94%예요` 문구를 제거했습니다.
- 우측 `재고 관리` 접이식 탭을 추가했습니다.
- 재고 관리 탭에서 상품별 옵션 재고를 직접 변경할 수 있습니다.
- 품절 옵션의 재고가 `0 → 1 이상`으로 변경되면 재입고 알림 발송 로그가 생성됩니다.
- 기존처럼 품절 옵션은 선택 박스에 담기지 않고 alert만 표시됩니다.
- 품절 옵션명은 `차콜헤더 : 재입고 알림은 톡톡에서 (품절)`로 표시됩니다.
- 톡톡문의 버튼은 실제 톡톡 페이지를 새 탭으로 엽니다.
- 하단에 셀러 안내용 Notion 문서 스타일 섹션, 데모 신청하기/문의하기/FAQ 모아보기 버튼, 티어 카드 hover 효과를 넣었습니다.

## 배포

ZIP 안의 내용을 저장소 루트에 덮어쓴 뒤 push하세요.

```bash
git add .
git commit -m "Add stock drawer and seller info section"
git push origin main
```
